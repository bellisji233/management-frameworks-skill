# 单元一详细内容：管理认知与策略提升

## 14方法详解

### 行动前（事前管理）
1. **设定期望**：明确角色职责、绩效标准、行为边界
2. **目标管理**：SMART目标设定、目标分解与对齐
3. **自由选法**：赋予下属选择方法的空间（在边界内）
4. **委派任务**：按能力意愿矩阵匹配任务难度
5. **一般指导**：日常方向性指引，非过度管控

### 行动中（过程管理）
6. **绩效对话**：定期1对1，聚焦进展与障碍
7. **问题解决（5步）**：定义问题→分析原因→生成方案→选择执行→复盘
8. **冲突对话**：事实陈述→影响说明→期望表达→共识寻求
9. **提供反馈**：SBI模型（情境→行为→影响）
10. **辅导对话**：GROW模型（目标→现状→选择→行动）

### 行动后（事后管理）
11. **认可激励**：及时、具体、公开的正向强化
12. **培养他人**：70-20-10原则（实践→他人→培训）
13. **汇报表达**：结论先行、数据支撑、建议可行
14. **复盘总结**：目标回顾→结果评估→原因分析→经验提炼

## 7S诊断详解

### 硬性三角（相对容易改变）
- **战略(Strategy)**：方向是否清晰，资源是否匹配
- **结构(Structure)**：汇报关系、职责划分、协作机制
- **制度(Systems)**：流程、规则、考核体系

### 软性四角（根植文化，较难改变）
- **共同价值观(Shared Values)**：团队信念与文化基因
- **风格(Style)**：领导风格与决策方式
- **员工(Staff)**：人才结构、能力分布
- **技能(Skills)**：团队核心能力所在

### 诊断使用场景
- 新团队接手：快速扫描7S，找到最薄弱环节
- 绩效下滑：从"症状"倒推是哪个S出了问题
- 变革推进：评估变革阻力来源

## 10潜力维度说明

用于识别、评估和培养高潜力人才：

| 维度 | 说明 |
|------|------|
| 领导意愿 | 是否真心想带人、负责任 |
| 价值观匹配 | 行为是否与公司文化一致 |
| 卓越绩效 | 持续高于预期的业绩表现 |
| 正直真诚 | 诚信、透明、言行一致 |
| 培养他人 | 主动分享、乐于辅导 |
| 获得结果 | 结果导向，不找借口 |
| 影响他人 | 无职权影响力，能推动协作 |
| 应变能力 | 面对不确定性时的稳定性 |
| 学习能力 | 快速吸收新知识、新技能 |
| 认知复杂度 | 处理模糊、复杂问题的能力 |
