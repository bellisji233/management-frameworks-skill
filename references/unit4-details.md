# 单元四详细内容：绩效评估与发展面谈

## ABC 分层面谈策略详解

按"业绩 × 能力/态度"把团队分成 A/B/C 三层，面谈打法完全不同。

### A 类（优秀）：认可 → 自评 → 挑战 → 需求 → 展望
重点是**留住 + 给更大舞台**。
- **认可**：先具体肯定成绩（用 STAR，不要泛泛"你很棒"）
- **自评**：让他先说，你会发现他对自己的判断和诉求
- **挑战**：给更大的目标/项目，满足成长需求
- **需求**：主动问他要什么（发展？话语权？资源？）
- **展望**：把他的未来和团队/公司绑定

### B 类（符合）：自评 → 激励 → 提升路径 → 支持
重点是**稳住 + 推动成长**。把"合格"往"优秀"推，关键是给清晰的提升路径和具体支持。

### C 类（欠佳）：坦诚 → 认可 → 诉求 → 改进 → 安排
重点是**明确边界 + 给机会**。坦诚指出差距（用数据），但同时认可可取之处、听его诉求、给出可执行的改进计划和时间节点。**边界和机会要同时给，不能只给压力。**

## 三类高难度员工处理

### A 类员工提离职 · 留才 4 步
1. **真诚挽留**：先表达重视，不要立刻谈条件
2. **倾听诉求**：搞清他真正要走的原因（钱？发展？关系？）
3. **分析判别**：判断诉求"可解决 / 不可解决"——不是所有人都该不计代价留
4. **有条件接一接**：可解决的，给明确的、能兑现的方案；不可解决的，体面放手

### B 类员工自我评价过高 · 3 步
1. **数据充分准备**：用事实说话，不是你的主观判断
2. **引导其自我发现差距**：让他对照数据自己得出结论，而不是你下结论
3. **共识具体行动计划**：落到可执行、可检验的动作

### C 类员工持续抱怨 · 5 法
*来源：实践框架，基于非暴力沟通（NVC）与认知行为原理*
倾听 → 视角转移 → 同理心回应 → 横向比较 → 升级对话（必要时引入上级）。
先让情绪被听见，再引导换个角度看，最后才是边界和升级。

## 积极面谈法
充分准备 → 营造安全感 → 双向对话 → **聚焦未来** → 明确行动。

> ⚠️ **反模式**：
> ① 绩效面谈只谈过去、只算账——重点应是"未来怎么改"；
> ② 对 C 类只给压力不给路径；
> ③ 因为某人能力强就容忍其态度问题（该"警示对话"还得上，见单元二⑨）。
